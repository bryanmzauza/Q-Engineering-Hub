## 🧩 Módulos

| Módulo | Status | Descrição |
|--------|--------|-----------|
| **Conversor de Unidades** | ✅ Pronto | Pressão, temperatura, vazão, viscosidade, massa, energia e mais |
| **Banco de Propriedades** | ✅ Pronto | Busca de substâncias com propriedades termodinâmicas |
| **Formulário** | ✅ Pronto | Resumo das principais fórmulas por disciplina |
| **Balanço de Massa** | 🔜 Em breve | Calculadora com correntes de entrada/saída e reações |
| **Balanço de Energia** | 🔜 Em breve | Cálculos com entalpia, Cp e trocadores |
| **Termodinâmica** | 🔜 Em breve | Equações de estado (ideal, Van der Waals, Peng-Robinson) |
| **Reatores Químicos** | 🔜 Em breve | CSTR, PFR, PBR — conversão, volume, tempo de residência |
| **Mecânica dos Fluidos** | 🔜 Em breve | Bernoulli, perda de carga, dimensionamento de bombas |
| **Transferência de Calor** | 🔜 Em breve | Condução, convecção, radiação, trocadores |
| **Transferência de Massa** | 🔜 Em breve | Difusão, torres de absorção |
| **Operações Unitárias** | 🔜 Em breve | Destilação (McCabe-Thiele), absorção, secagem |

## 🤝 Como contribuir

1. Faça um **fork** do repositório
2. Crie uma branch para sua feature:
   ```bash
   git checkout -b feature/minha-feature
   ```
3. Faça commit das suas alterações:
   ```bash
   git commit -m "feat: adiciona nova feature"
   ```
4. Faça push para a branch:
   ```bash
   git push origin feature/minha-feature
   ```
5. Abra um **Pull Request**
